Introduction

The nonLinearImageFilter application allows one to apply either linear Gaussian filter or non-linear Gaussian filter to grayscale images. The images can loaded from TIFF, PNG, GIF and JPEG grayscale files. The program also supports generation of test images. The filtered images can be saved as grayscale TIFF, PNG and GIF files or as uncompessed AVI or QuickTime videos.

Description

Mathematical model

The linear Gaussian filtering is performed by solving the linear diffusion equation for image brightness b with a constant diffusion coefficient η:

∂b/∂n=∇(η∇b),

where n is filtering step and the gradient is in the image coordinates.

In non-linear Gaussian filter the diffusion coefficient is Gaussian function of the gradient of image brightness b:

η=η0×exp(−(grad(b)/σ)2),

where σ is the non-linear coefficient.

Filtering procedure

Use Initialize button to either generate a new test image or load a new image from file. Then choose filter parameters: diffusion coefficient, non-linear coefficient and number of filtering steps. Tick Non-linear filtering box if non-linear filtering is required. Press Filter button to begin filtering. Observe as the filtering is going by looking at the Progress bar. The process can be stopped by pressing Stop button. The filtering can be continued after adjusting parameters. The sequence of filtered images can be viewed by moving the slider in Controls panel. The currently visible image can be saved by using File->Save current image... menu item. The full image sequence can be save as AVI or QuickTime video by using File->Save as video... menu option.

Options

Options->Test image options.. allows one to specify: the test image width and height, the signal level outside the square, the noise level and the relative scale of the square. Options->Filter options... allows one to enter the relative numerical precision of non-linear filtering.
